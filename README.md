# IRAN DISCORD

Official Discord bot for the IRAN DISCORD community and Iranian server directory.

- Website: https://www.irandiscord.com
- Server directory: https://www.irandiscord.com/servers
- Dashboard: https://www.irandiscord.com/dashboard
- Community: https://discord.gg/ZMv3KdsA5A

Free to use on your own Discord server.

---

## Features

### Dashboard
Manage the bot from a simple web panel on irandiscord.com. No need to dig through slash commands for everyday settings.

### Live & upload alerts
Get Twitch, YouTube, Kick, and Aparat alerts in your server when a stream goes live or a new video is posted.

### Giveaways
Run giveaways with custom embeds and full control from the dashboard.

### Coming soon
More tools are on the way. New features show up on the website bot section as they ship.

## Self-host (optional)

If you want to run your own copy:

1. Install Go 1.22+
2. Copy `.env.example` to `.env` and fill in your values
3. Build and run:

```bash
go build -o bin/iran-discord-stats ./cmd/bot
./bin/iran-discord-stats
```

Windows:

```bat
go build -o bin\iran-discord-stats.exe .\cmd\bot
.\bin\iran-discord-stats.exe
```

## فارسی

بات رسمی **ایران دیسکورد** برای کامیونیتی و لیست سرورهای ایرانی.

- سایت: https://www.irandiscord.com
- لیست سرورها: https://www.irandiscord.com/servers
- داشبورد: https://www.irandiscord.com/dashboard
- سرور رسمی: https://discord.gg/ZMv3KdsA5A

استفاده از بات روی سرور خودت رایگانه.

### قابلیت‌ها

**داشبورد اختصاصی**  
از یه پنل راحت روی سایت، بات رو خودت مدیریت و تنظیم کن.

**اعلان لایو و آپلود**  
اعلان‌های توییچ، یوتیوب، کیک و آپارات رو مستقیم تو سرورت داشته باش؛ لایو شد یا ویدیو اومد، همون لحظه خبر می‌ده.

**گیواوی**  
گیواوی حرفه‌ای با امبد سفارشی و مدیریت کامل از داشبورد.

**به‌زودی**  
قابلیت‌های تازه کم‌کم اضافه می‌شن. جزئیات رو تو بخش بات سایت می‌بینی.

### لینک‌ها

- افزودن بات: از سایت ایران دیسکورد (بخش بات / داشبورد)
- پشتیبانی: سرور رسمی ایران دیسکورد
